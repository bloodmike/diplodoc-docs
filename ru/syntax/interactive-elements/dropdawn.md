# Дропдаун

Используйте дропдаун, чтобы получить удобный выпадающий список, если элементов со скрытым контентом слишком много.

```markdown
{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2

  Контент для пункта 2.

{% endlist %}
```

**Результат:**

{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2

  Контент для пункта 2.

{% endlist %}
